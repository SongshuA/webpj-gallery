## 项目部署与运行

编译前端静态文件(webpack)
```
npm start
```

开启测试服务器(webpack-dev-server)
```
npm run server
```
*服务器地址：[http://localhost:8080/gallery/](http://localhost:8080/gallery/)*


开启WebSocket服务器（热部署）
```
npm run socket-dev
```

开启WebSocket服务器
```
npm run socket
```
